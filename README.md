# Dashboard Profesional Analytics

Una aplicación web profesional construida con Dash para análisis de datos en tiempo real.

## Características

- 📊 Dashboard interactivo con múltiples gráficos
- 📈 KPIs animados y métricas en tiempo real
- 🎨 Diseño responsive y profesional
- 🔍 Filtros avanzados por fecha, categoría y región
- 📱 Compatible con dispositivos móviles
